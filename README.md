# Aumuc - Automotive Units Converter
A PHP library to convert car related units, like `PS` to `kW` or `mph` to `km/h`.

## Currently supported

### Force
* Nm
* lbs ft

### Power
* PS
* kW
* bhp

### Speed
* mph
* km/h

### Volume
* ccm
* cu in

### Weight
* kg
* lb

⚠ Warning: This is a very early version. Don't use in production. ⚠
